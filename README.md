# ICT practice

Hands-on practice material for the Rady MSBA **ICT** workshop practice
sessions. Everything here is about getting comfortable with the RSM-MSBA
computing environment: the terminal, file paths, folder structure, and
folders that live on GitHub.

## Get it

From a VS Code terminal in `~/rsm-msba`:

```bash
cd ~/rsm-msba
rsm-clone git@github.com:rsm-msba-26-27/ict-practice.git
cd ict-practice
```

## What's here

| Where | What | Used in |
|---|---|---|
| `check-setup.sh` | pass/fail checklist that your environment is fully set up | Practice 01, part A |
| `hunt/` | **The Secret Taco Stand** — a terminal treasure hunt (start at `hunt/START.txt`) | Practice 01, part C |
| `messy-project/` + `MAKEOVER.md` | A disaster of a project folder for you to reorganize | Practice 01, part D |
| `check-structure.sh` | Grades your makeover | Practice 01, part D |
| `paths-rescue/` | A script + Quarto report that only run from the right folder | Practice 02, part B |
| `merge-conflict/` | Create and resolve a merge conflict, on purpose | Practice 02, part C |

The full session pages are on the ICT course site (Practice 01 and
Practice 02 in the navbar).

## House rules

- This repo is **read-only practice material** — you never push here. Copy
  things out (`MAKEOVER.md` shows how) and do your work in your own folders
  and your own repo.
- Refresh it any time with `git pull` from inside the folder.
